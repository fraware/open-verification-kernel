# OVK v0.1 Preflight Addendum

Maintainers should run the release preflight before tagging v0.1.

Preflight script:

```text
scripts/release_preflight.py
```

The preflight combines:

- release metadata consistency;
- command-surface consistency;
- local release smoke checks.

This addendum supplements `docs/V0_1_TAGGING_CHECKLIST.md`.
